# ColorName

## Usage
### SwiftUI
```swift
let myColorName = getName(for: Color.red)
print(myColorName)
```

### UIKit
```swift
let myColorName = getName(for: UIColor.red)
print(myColorName)
```

### CGColor
```swift
let myColorName = getName(for: CGColor.white)
print(myColorName)
```
